# skill
Codex skill packages that drive installation-time and runtime bridge workflows.
Skills stay lightweight: they explain how Codex should invoke the installed bridge rather than duplicating bridge logic.
The first skill is Telegram-specific, but the repository name intentionally leaves room for more chat adapters later.
