# docs
Documentation root for architecture decisions, diagrams, and release-quality verification notes.
The most important documents in this repo are the folder `ARCH.md` files, ADRs, and the C4 workspace.
Documentation here is expected to evolve in lockstep with module boundaries rather than trailing behind code.
一旦我所属的文件夹有所变化，请更新我。

| file name | position | function |
| --- | --- | --- |
| `adr/` | decision records | Stores architectural decisions and trade-off history. |
| `architecture/` | system model docs | Stores the C4 workspace and related architecture material. |
| `green-checkpoints.md` | delivery ledger | Records all-green checkpoints after verification passes. |
